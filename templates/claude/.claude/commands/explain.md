# Explain Command

Explain how the selected code works in detail.

## Usage

```
/explain [code selection]
```

## Explanation Format

Provide a clear explanation covering:

1. **Purpose** — What does this code do?
2. **How It Works** — Step-by-step breakdown
3. **Key Concepts** — Important patterns or techniques used
4. **Dependencies** — What does it rely on?
5. **Usage Example** — How to use it

## Output Template

```markdown
## Explanation: [code summary]

### Purpose

[What this code accomplishes]

### How It Works

1. [Step 1]
2. [Step 2]
3. [Step 3]

### Key Concepts

- **[Concept]**: [Explanation]

### Example Usage

\`\`\`typescript
// Example of how to use this code
\`\`\`

### Related

- [Link to relevant docs or related code]
```
