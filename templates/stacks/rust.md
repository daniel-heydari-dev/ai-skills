## Stack: Rust

- Language: Rust
- Build Tool: Cargo

## Commands

```bash
# Development
cargo run

# Build
cargo build --release

# Test
cargo test

# Lint
cargo clippy
```

## Conventions

- Follow Rust API guidelines
- Use `?` operator for error handling
- Prefer `impl Trait` over generics when possible
- Use `#[derive]` for common traits
